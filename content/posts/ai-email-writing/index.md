## AI 이메일 작성의 장점

업무 이메일은 내용보다 표현에서 시간이 많이 걸릴 때가 많습니다. AI를 활용하면 정중한 표현, 짧은 회신, 일정 변경 요청, 사과 메일 초안을 빠르게 만들 수 있습니다.

다만 이름, 날짜, 금액, 첨부파일 같은 사실 정보는 AI가 실수할 수 있으므로 발송 전 반드시 확인해야 합니다.

## 기본 프롬프트

```text
너는 비즈니스 이메일 작성 도우미야.
아래 내용을 정중하고 자연스러운 업무 메일로 바꿔줘.
받는 사람: 외부 파트너사 담당자
목적: 일정 변경 요청
톤: 정중하지만 너무 딱딱하지 않게
길이: 5문장 이내

포함할 내용:
- 기존 미팅을 다음 주로 변경하고 싶음
- 화요일 오후 또는 수요일 오전 가능
- 불편을 드려 죄송하다는 표현
```

## 요청 메일 프롬프트

```text
아래 요청 사항을 상대방이 부담스럽지 않게 느끼도록 정중한 이메일로 작성해줘.
요청 내용은 명확하게 쓰고, 마지막에는 회신을 부탁하는 문장을 넣어줘.
```

## 사과 메일 프롬프트

```text
아래 상황을 사과 이메일로 작성해줘.
변명처럼 들리지 않게 하고, 원인, 사과, 해결 방안, 재발 방지 순서로 정리해줘.
```

## 회신 메일 프롬프트

```text
아래 메일에 대한 답장을 작성해줘.
상대방의 요청을 확인했다는 내용과 다음 액션을 포함해줘.
문장은 짧고 명확하게 작성해줘.
```

## 최종 확인 체크리스트

- 이름과 회사명이 맞는지 확인
- 날짜와 시간 확인
- 첨부파일 언급이 맞는지 확인
- 너무 과장되거나 딱딱한 표현은 없는지 확인
- 회사 내부 정보가 포함되지 않았는지 확인

## 정리

AI 이메일 작성은 빈 문서에서 시작하는 부담을 줄여줍니다. 받는 사람, 목적, 톤, 핵심 내용을 분명하게 입력하고 마지막 검토만 꼼꼼히 하면 업무 시간을 줄이는 데 도움이 됩니다.
