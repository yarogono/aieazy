## 업무에서 ChatGPT를 쓸 때의 기준

직장인이 ChatGPT를 쓸 때는 빠른 초안 작성과 정리가 핵심입니다. 완성된 결과를 그대로 복사하기보다, 첫 초안을 만들고 사람이 최종 검토하는 방식이 가장 현실적입니다.

특히 회사 내부 정보, 고객 개인정보, 계약 조건처럼 민감한 내용은 입력하지 않는 것이 좋습니다.

## 이메일 작성 프롬프트

```text
너는 비즈니스 이메일 작성 도우미야.
아래 내용을 정중한 업무 이메일로 바꿔줘.
상대방은 외부 파트너사 담당자야.
톤은 부드럽지만 명확하게 작성해줘.

내용:
- 미팅 일정 변경 요청
- 다음 주 화요일 또는 수요일 가능
- 불편을 드려 죄송하다는 문장 포함
```

## 보고서 정리 프롬프트

```text
아래 메모를 팀장에게 공유할 보고서 형식으로 정리해줘.
구성은 배경, 현재 상황, 문제점, 제안, 다음 액션 순서로 작성해줘.
문장은 간결하게 써줘.
```

## 회의록 요약 프롬프트

```text
아래 회의 메모를 회의록으로 정리해줘.
출력 형식:
- 핵심 결정사항
- 논의 내용
- 담당자별 할 일
- 다음 회의 전 확인할 내용
```

## 기획안 초안 프롬프트

```text
너는 서비스 기획자야.
주제는 신규 AI 콘텐츠 허브 페이지 개선안이야.
목표, 문제 정의, 개선 방향, 기대 효과, 우선순위를 포함해서 기획안 초안을 작성해줘.
```

## 엑셀 정리 프롬프트

```text
아래 표 데이터를 보고 어떤 기준으로 정리하면 좋을지 제안해줘.
필요하면 엑셀 함수나 피벗테이블 기준도 함께 알려줘.
초보자도 따라 할 수 있게 단계별로 설명해줘.
```

## 정리

업무용 ChatGPT 프롬프트는 멋진 표현보다 명확성이 중요합니다. 상대방, 목적, 톤, 형식, 분량을 함께 적으면 이메일과 보고서 품질이 훨씬 안정적으로 나옵니다.
