챗지피티에서 원하는 답변을 얻으려면 역할, 상황, 작업, 출력 형식을 함께 적어야 합니다. “알려줘”보다 “너는 [역할]이고, [상황]에서 [작업]을 [형식]으로 해줘”처럼 쓰면 결과가 훨씬 좋아집니다.

## 챗지피티 프롬프트 추천 작성법과 입력 예시 먼저 보기

| 요소 | 입력 방법 |
| --- | --- |
| 역할 | “너는 10년 차 [분야] 전문가야”처럼 답변 관점을 정합니다. |
| 상황 | 왜 필요한지, 누구를 위한 답변인지 배경을 알려줍니다. |
| 작업 | 요약, 비교, 작성, 수정, 아이디어 제안처럼 할 일을 명확히 씁니다. |
| 형식 | 표, 목록, 단계별 설명, 이메일 초안처럼 출력 형태를 지정합니다. |
| 톤 | 친근하게, 전문적으로, 짧게, 자연스럽게 같은 말투를 정합니다. |

바로 쓰려면 이렇게 입력하면 됩니다. “너는 [역할]이야. 나는 [상황]이야. [작업]을 해주고, 결과는 [형식]으로 정리해줘.”

## 완벽한 답변을 유도하는 R-C-S-T 프롬프트 공식

챗지피티는 문맥을 명확히 정의해 줄 때 가장 정교한 답변을 출력합니다.
아래 4가지 핵심 요소를 갖춰서 질문하면 답변의 질이 200% 이상 향상됩니다.

### R (Role)과 C (Context)로 인공지능에 맥락 부여하기

답변의 방향성을 잡아주는 첫 단추는 역할과 배경을 정의하는 것입니다.
AI에게 특정 분야의 전문가라는 가면을 씌워주고,  
왜 이 질문을 하는지 현재 상황을 상세히 설명해야 합니다.

예를 들어 단순히 "마케팅 문구 써줘"가 아니라, "너는 10년 차 소셜 미디어 크리에이터야(Role).  
새해를 맞아 시간이 부족한 2030 직장인을 타겟으로 홈트레이닝 상품을 홍보하려고 해(Context)"와 같이 구체적인 환경을 제공하는 방식입니다.

### S (Task)와 T (Format/Tone)로 출력 결과 제어하기

원하는 형태의 결과물을 얻기 위해서는 작업 명령과 형식을 지정해야 합니다.

수행해야 할 구체적인 작업 내용을 명시하고, 최종 출력의 형태와 어조를 명확히 제한해야 합니다.

"릴스 콘텐츠 아이디어 5가지를 제안해줘(Task)"라고 명령한 뒤,  
"핵심 내용이 한눈에 보이도록 표(Table) 형태로 정리하고, 활기찬 톤앤매너를 유지해줘(Format/Tone)"와 같이  
제약 조건을 설정하면 불필요한 재질문을 줄일 수 있습니다.

## 원하는 답변을 얻는 입력 방법 예시 및 비교

동일한 주제를 질문하더라도 막연한 질문(나쁜 예)과 공식을 적용한 질문(좋은 예)은 결과물에서 큰 차이를 보입니다.

### 마케팅 기획 및 아이디어 도출 상황

타겟 독자의 공감을 얻고 저장과 공유를 유도하는 아이디어를 얻는 입력 방식입니다.

* **나쁜 예:** "인스타그램에 올릴 만한 홈트레이닝 콘텐츠 아이디어 좀 줘."
* **좋은 예 (추천 작성법):**
> **역할**: 너는 퍼스널 트레이닝(PT) 전문 인스타그램 계정을 운영하는 소셜 미디어 크리에이터야.
> **상황**: 새해를 맞아 '집에서 혼자 운동하는 2030 직장인'들을 타겟으로 삼고 있어. 이들은 시간이 없고 의지가 약하다는 특징이 있어.
> **작업**: 이들의 공감을 얻고 저장/공유를 유도할 수 있는 인스타그램 릴스(Reels)용 콘텐츠 아이디어 5가지를 제안해줘.
> **출력 형식**: 콘텐츠 제목, 핵심 내용, 기대 효과를 포함해서 표(Table) 형태로 정리해줘. 톤앤매너는 활기차고 동기부여를 주는 느낌으로 해줘.



### 정보 요약 및 학습 상황

어려운 과학 이론이나 개념을 눈높이에 맞게 쉬운 비유로 학습하는 입력 방식입니다.

* **나쁜 예:** "양자역학이 뭔지 쉽게 설명해줘."
* **좋은 예 (추천 작성법):**
> **역할**: 너는 복잡한 과학 이론을 초등학생도 이해할 수 있게 가르치는 스타 강사야.
> **상황**: 과학 지식이 전혀 없는 사람에게 양자역학의 핵심 개념인 '중첩'을 설명해야 해.
> **작업**: 양자역학의 개념을 일상적인 비유(예: 동전 던지기 등)를 들어서 설명해줘.
> **출력 형식**: 전체 글자 수는 공백 포함 500자 이내로 하고, 마지막에는 핵심 내용을 3줄 요약으로 마무리해줘. 친근한 대화체(~요 체)를 사용해줘.



### 코딩 및 엑셀 매크로 요청 상황

정제되지 않은 데이터를 원하는 규칙에 맞게 자동으로 분리하는 입력 방식입니다.

* **나쁜 예:** "엑셀에서 이름 분리하는 법 알려줘."
* **좋은 예 (추천 작성법):**
> **역할**: 데이터 분석 및 엑셀 자동화 전문가.
> **상황**: 현재 A열에 '홍길동(인사팀)', '이영희(마케팅팀)' 형태로 이름과 부서가 한 셀에 합쳐져 있어.
> **작업**: 가로 열을 기준으로 이름은 A열에, 부서명(괄호 제외)은 B열에 자동으로 분리되도록 만드는 엑셀 수식(또는 VBA 코드)을 알려줘.
> **출력 형식**: 수식을 먼저 텍스트 코드로 보여주고, 수식의 작동 원리를 단계별(Step-by-Step)로 초보자가 따라 할 수 있게 가이드라인을 적어줘.



## 답변 퀄리티를 비약적으로 높이는 치트키 문구

프롬프트 맨 마지막에 특정 문장 한 줄을 추가하는 것만으로도 AI의 추론 능력을 극대화할 수 있습니다.

### 논리적 추론과 역질문을 유도하는 팁

AI가 성급하게 결론을 내리지 않고 깊이 있게 생각하도록 유도하는 명령어입니다.

"단계별로 생각해서 논리적으로 답변해줘 (Think Step-by-Step)"라는 문구를 추가하면,  
복잡한 연산이나 기획 문제에서 중간 과정을 꼼꼼하게 검토하며 오답률을 낮춥니다.

또한 "답변을 작성하기 전에, 더 정확한 결과를 위해 나에게 부족한 정보가 있다면 먼저 질문을 던져줘"라고 덧붙이면, 
질문의 모호함을 해소하기 위해 AI가 역으로 맥락을 좁혀오는 고난도 대화가 가능해집니다.

### AI 특유의 기계적인 말투 제거하기

블로그나 이메일 작성 시 사람이 쓴 것처럼 자연스러운 문장을 얻는 방법입니다.

"AI 특유의 상투적인 표현(예: ~에 대해 알아보겠습니다, 첫째로, 결론적으로 등)은 절대 사용하지 마"라는 제약을 주면,  
판에 박힌 서론과 결론 양식을 즉시 배제하고 핵심 본론으로 직행하는 깔끔한 텍스트를 얻을 수 있습니다.
