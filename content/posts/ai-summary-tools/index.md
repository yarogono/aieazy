## AI 요약 도구가 유용한 상황

AI 요약은 긴 글을 빠르게 파악해야 할 때 도움이 됩니다. 뉴스 기사, 보고서, 회의 메모, 강의 자료, 블로그 글, 이메일 스레드처럼 읽을 양이 많을 때 특히 유용합니다.

다만 요약은 원문을 줄이는 과정이기 때문에 중요한 조건이나 예외가 빠질 수 있습니다. 계약서, 의료, 법률, 투자처럼 중요한 판단이 필요한 내용은 반드시 원문을 다시 확인해야 합니다.

## 기본 요약 프롬프트

```text
아래 글을 요약해줘.
구성:
- 핵심 요약 5줄
- 중요한 용어
- 독자가 꼭 알아야 할 내용
- 추가로 확인해야 할 질문
```

## 회의록 요약 프롬프트

```text
아래 회의 메모를 정리해줘.
출력 형식:
- 결정된 내용
- 논의 중인 내용
- 담당자별 할 일
- 마감일
- 다음 회의에서 확인할 질문
```

## 문서 요약 프롬프트

```text
너는 문서 요약 전문가야.
아래 문서를 읽고 실무자가 빠르게 이해할 수 있게 정리해줘.
요약은 너무 짧게 하지 말고, 중요한 조건과 예외를 따로 표시해줘.
```

## 표로 정리하는 프롬프트

```text
아래 내용을 표로 정리해줘.
열은 항목, 핵심 내용, 중요도, 확인할 점으로 만들어줘.
중요도는 높음, 보통, 낮음으로 구분해줘.
```

## AI 요약을 쓸 때 주의할 점

- 원문에 없는 내용을 AI가 추가하지 않았는지 확인
- 숫자, 날짜, 이름은 원문과 대조
- 중요한 조건과 예외가 빠지지 않았는지 확인
- 민감한 내부 문서는 입력하지 않기
- 최종 판단은 원문 확인 후 하기

## 정리

AI 요약 도구는 시간을 줄여주는 강력한 보조 도구입니다. 핵심 요약, 표 정리, 할 일 추출처럼 목적을 나눠서 요청하면 더 정확하고 실용적인 결과를 얻을 수 있습니다.
