## 학생이 ChatGPT를 써도 되는 방식

ChatGPT는 공부를 대신해주는 도구가 아니라 이해를 도와주는 도구로 쓰는 것이 좋습니다. 특히 어려운 개념 설명, 긴 글 요약, 복습 문제 만들기, 발표 아이디어 정리에 유용합니다.

과제에 사용할 때는 학교나 수업의 AI 사용 규칙을 먼저 확인해야 합니다. AI가 만든 글을 그대로 제출하는 방식은 문제가 될 수 있습니다.

## 개념 설명 프롬프트

```text
너는 중학생도 이해할 수 있게 설명하는 선생님이야.
아래 개념을 쉬운 예시로 설명해줘.
마지막에는 핵심 요약 3줄과 확인 문제 3개를 만들어줘.

개념: 머신러닝
```

## 긴 글 요약 프롬프트

```text
아래 글을 공부용으로 요약해줘.
구성:
- 핵심 내용 5줄
- 중요한 용어 정리
- 시험에 나올 만한 질문 5개
- 헷갈리기 쉬운 부분
```

## 암기용 퀴즈 프롬프트

```text
아래 내용을 바탕으로 객관식 문제 5개와 주관식 문제 5개를 만들어줘.
정답과 해설도 함께 작성해줘.
난이도는 쉬움, 보통, 어려움이 섞이게 해줘.
```

## 발표 준비 프롬프트

```text
주제는 AI가 일상생활에 미치는 영향이야.
3분 발표용 대본을 만들어줘.
구성은 도입, 본론 3개, 결론 순서로 작성해줘.
학생이 말하는 것처럼 자연스럽게 써줘.
```

## 정리

학생에게 ChatGPT는 답을 대신 써주는 도구보다 공부 과정을 빠르게 정리해주는 도구로 더 유용합니다. 설명, 요약, 퀴즈, 발표 구조처럼 학습을 돕는 방향으로 쓰면 효과가 좋습니다.
